新規キャラ追加
キャラ1人分のJSONを貼れ（オブジェクト）。配列でも先頭1件だけ拾う。
貼り付け追加のルール：
・series / genre / colorTag は空でもOK
・genderがfemale/male以外なら other 扱い
・idはカテゴリ反映時に自動採番になる（上書き）
・imageは "images/xxx.png" 推奨
追加先の子jsonを選べ。seriesはその子json先頭キャラのseriesを自動で入れる（無ければ子名）。
「空キャラ」は最低限の形で追加する：
name: "新規キャラ" / gender: other / series: 自動 / genre: "不明"
あとは右側で編集して反映。
